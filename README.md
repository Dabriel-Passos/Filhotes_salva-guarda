# Filhotes_salva-guarda
Projeto para causa de adoção de animais, feita para o prosel 26.1 da Cimatec JR.

Me chamo Gabriel de Araujo Linhares Passos, estou no 1º semestre de engenharia de computação no SENAI Cimatec, escolhi o tema de resgate e adoção de animais para o site da etapa técnica do prosel 26.1. Como um primeiro contato com HTML e CSS achei bem interessante e desafiador, utilizei de ambas com um pouco de java script dentro do HTML.
